# 부록 A: 용어집

openpilot 관련 주요 용어 정리

## A

**ACC (Adaptive Cruise Control)**
- 앞차와의 거리를 유지하며 속도를 자동 조절하는 기능

**Actuators**
- 차량을 제어하는 물리적 장치 (조향, 가속, 브레이크)

## C

**CAN (Controller Area Network)**
- 차량 내부 통신 프로토콜

**Cap'n Proto**
- openpilot이 사용하는 직렬화 라이브러리

**Cereal**
- openpilot의 메시징 시스템 (Cap'n Proto 기반)

**controlsd**
- openpilot의 메인 제어 프로세스

## D

**DBC (Database CAN)**
- CAN 메시지 정의 파일 형식

**Device (디바이스)**
- comma three, comma 3X 등 openpilot 실행 장치

## F

**Fingerprinting**
- 차량 모델을 자동으로 감지하는 기술

## L

**LKA (Lane Keeping Assist)**
- 차선 유지 보조 기능

**Lateral Control**
- 횡방향(조향) 제어

**Longitudinal Control**
- 종방향(속도) 제어

## M

**modeld**
- AI 비전 모델을 실행하는 프로세스

**MPC (Model Predictive Control)**
- 예측 제어 알고리즘

## P

**panda**
- comma.ai의 CAN 인터페이스 하드웨어

**plannerd**
- 경로 계획 프로세스

**PubSub**
- Publish-Subscribe 메시징 패턴

## S

**SCons**
- openpilot의 빌드 시스템

**SNPE (Snapdragon Neural Processing Engine)**
- Qualcomm의 AI 추론 엔진

**supercombo**
- openpilot의 메인 비전 모델

## T

**Torque**
- 조향 토크 (조향력)

## Z

**ZMQ (ZeroMQ)**
- openpilot 내부 통신에 사용되는 메시징 라이브러리
